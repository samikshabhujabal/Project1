🔧 CUSTOMIZATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Change Primary Color:
→ Edit: static/css/styles.css
→ Find: --ai-primary: #0066FF;
→ Change to your preferred blue

Adjust Glow Intensity:
→ Find: --neon-shadow: 0 0 20px rgba(0, 102, 255, 0.5);
→ Increase: 0 0 40px rgba(0, 102, 255, 0.8);
→ Decrease: 0 0 10px rgba(0, 102, 255, 0.3);

Change Background Opacity:
→ Find: opacity: 0.3;
→ More visible: 0.5
→ Less visible: 0.2

Disable Animations:
→ Comment out animation lines
→ Or add: animation: none !important;
