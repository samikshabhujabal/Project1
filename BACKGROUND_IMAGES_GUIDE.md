## 📋 Download Instructions

### Step 1: Visit Image Sources
1. Go to Unsplash.com or Pexels.com
2. Search using the terms above
3. Select high-resolution images (1920x1080 or larger)

### Step 2: Download Images
1. Click on the image you like
2. Click "Download" button
3. Choose the largest size available (usually "Original" or "Large")

### Step 3: Save to Project
Save downloaded images to: `static/img/backgrounds/`

Recommended naming:
- `bg-landing.jpg` or `bg-landing.png`
- `bg-dashboard.jpg`
- `bg-emergency.jpg`
- `bg-donor.jpg`
- `bg-medical.jpg`
- `bg-profile.jpg`
- `bg-awareness.jpg`
- `bg-camps.jpg`
- `bg-about.jpg`
- `bg-services.jpg`

## 🎨 Image Specifications

### Recommended Dimensions:
- **Width:** 1920px minimum
- **Height:** 1080px minimum
- **Format:** JPG (smaller file size) or PNG (better quality)
- **File Size:** Keep under 500KB for performance

### Image Style Guidelines:
- **Brightness:** Slightly desaturated or with overlay capability
- **Focus:** Soft focus or blurred backgrounds work best
- **Colors:** Should complement the color palette (reds, blues)
- **Contrast:** Not too high contrast (text needs to be readable)

## ⚡ Performance Tips

1. **Optimize images** before using:
   - Use tools like TinyPNG (https://tinypng.com)
   - Compress to 70-80% quality
   - Keep file size under 500KB

2. **Use WebP format** for better compression:
   - Convert JPG/PNG to WebP
   - Provide fallback for older browsers

3. **Lazy loading:**
   - Images load only when needed
   - Improves initial page load time
